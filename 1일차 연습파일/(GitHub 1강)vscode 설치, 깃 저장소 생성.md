
깃허브(GitHub) 저장소는 어떻게 만드는 거야? 깃허브는 코드를 저장하는 **온라인 공간**인데, 여기서 **새로운 **저장소**(repository)**를 만들고 이름을 지어주면 돼.
## 1. 코딩 준비물 설치하기 

- 코딩을 하려면 몇 가지 프로그램이 필요해.
  - VS 코드 (Visual Studio Code): 코딩할 때 쓰는 프로그램이야. 
  - 검색창에 'VS 코드'라고 치고 나오는 걸 설치하면 돼. 
  - 깃허브 (GitHub): 코드를 저장하는 온라인 공간이야. 
  - 깃허브 홈페이지에 들어가서 회원가입을 해야 해. 
  - 구글 계정으로도 쉽게 가입할 수 있어. 

## 2. VS 코드 설치 과정 자세히 알아보기 

- VS 코드를 설치할 때는 몇 가지 주의할 점이 있어. 
  - 먼저 VS 코드 홈페이지에서 '다운로드 포 윈도우(Download for Windows)'를 눌러 설치 파일을 받아. 
  - 다운로드된 파일을 열어서 설치를 시작해. 
  - 설치 창이 뜨면 '동의합니다'를 선택하고 '다음'을 눌러. 
  - 중요한 건, '코드를 열기 작업'과 관련된 두 가지 항목을 **꼭 체크**해야 해. 
  - 이걸 체크 안 하면 나중에 코딩할 때 문제가 생길 수 있어. 
  - 나머지 설정은 그대로 두고 '다음'을 누른 다음 '설치'를 누르면 돼. 

## 3. 깃허브는 코드를 보관하는 온라인 창고! 

- 깃허브는 네가 짠 코드를 저장하고 관리하는 **온라인 공간**이야. 
  - 마치 네가 만든 작품들을 모아두는 개인 창고 같은 거지. 
  - 이 창고에 코드를 저장해두면 언제 어디서든 꺼내 쓸 수 있어. 
  - 다른 사람들과 함께 코드를 만들 때도 아주 유용해.

## 4. 깃허브에 나만의 코드 창고 만들기 (저장소 생성) 

- 깃허브에 로그인했다면, 이제 네 코드를 보관할 **창고** (저장소, **리포지토리**)를 만들어야 해. 
  - 깃허브 대시보드에서 '새로운 저장소 만들기' 버튼을 찾아 눌러. 
  - 창고 이름을 정할 때는 **영어와 숫자**만 사용해야 해. 
  - 컴퓨터는 한글을 잘 못 알아듣거든. 
  - 예를 들어, '수업-데이1'처럼 이름을 지을 수 있어. 
  - 이름을 정했으면, 초록색 버튼을 눌러서 창고를 만들면 돼. 
  - 이렇게 하면 네 코드를 저장할 온라인 창고가 생긴 거야! 



## 5. 깃허브 페이지로 웹사이트 만들기 (공짜 도메인) 

- 깃허브 저장소를 이용하면 간단한 웹사이트를 **공짜**로 만들 수 있어. 
  - 보통 웹사이트 주소(도메인)는 돈을 내야 하지만, 깃허브는 이걸 무료로 제공해 줘. 
  - 대신 아주 복잡한 웹사이트는 만들 수 없어. 
- 웹사이트를 만들려면 몇 가지 설정을 해야 해. 
  - 저장소 설정(Settings)에 들어가서 '페이지(Pages)' 메뉴를 찾아. 
  - '브랜치(Branch)' 설정에서 '메인(main)'을 선택하고 저장하면 돼. 
  - 만약 '메인'이 바로 안 보이면, '소스(Source)'에서 '액션(Action)'을 눌렀다가 다시 '브랜치'로 바꾸면 '메인'이 나타날 거야. 
  - 이 설정이 끝나면, 네 코드를 올렸을 때 웹사이트 주소가 자동으로 생길 거야. 

## 6. 깃허브에 파일 올리기 준비 

- 이제 네가 만든 코드를 깃허브 창고에 넣을 준비를 해야 해. 
  - 깃허브 저장소 페이지에서 '코드(Code)' 버튼을 누르면 파일을 올릴 수 있는 공간이 나와. 
  - 원래는 '새 파일(New file)' 버튼으로 바로 만들 수 있지만, 지금은 깃허브 시스템에 문제가 있어서 잘 안 될 수도 있어. 
  - 그럴 때는 '파일 올리기(Upload files)' 버튼을 눌러서 파일을 올려야 해. 
  - 파일을 하나 임시로 올려두고 나서 나머지 작업을 이어서 할 거야. 

## 7. 코딩 파일 만들기 (메모장 활용) 

- 코딩 파일을 만들려면 먼저 컴퓨터에 파일 탐색기를 열어야 해. 
  - 키보드에서 윈도우 키(창문 모양)와 'E' 키를 동시에 누르면 파일 탐색기가 열려. 
- 파일을 만들 위치는 '다운로드' 폴더가 좋아. 
  - '다운로드' 폴더에 오늘 날짜로 새 폴더를 만들고 이름을 '독스(docs)'라고 지어. 
  - 이 '독스' 폴더 안에 코딩 파일을 만들 거야. 
- 코딩 파일을 가장 빠르게 만드는 방법은 **메모장**을 이용하는 거야. 
  - '독스' 폴더 안에서 마우스 오른쪽 버튼을 누르고 '새로 만들기' -> '메모장'을 선택해. 
  - 중요한 건, 파일 이름 뒤에 '.txt'라는 확장자가 보여야 해. 
  - 만약 안 보인다면, 파일 탐색기 메뉴에서 '보기'를 누르고 '파일 확장명'을 체크하면 돼. 
  - 이 메모장 파일을 나중에 HTML 파일로 바꿀 거야.
