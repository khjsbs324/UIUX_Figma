
깃(Git)을 어떻게 설치해야 할까? 윈도우 64비트 버전을 다운로드해서 **기본값으로 **쭉 설치하면 돼.
## 1. 깃(Git) 설치, 어렵지 않아! 
<gen-svg-visual description="A simple flowchart showing the Git installation process: Start -> Download <keyword data-source-id=" />

- 깃(Git)을 설치하는 건 마치 새로운 게임을 컴퓨터에 까는 거랑 똑같아. 
  - 복잡하게 생각할 필요 없어. 그냥 시키는 대로 하면 돼. 
- 윈도우 컴퓨터를 쓴다면, **윈도우 64비트 버전**을 다운로드하면 돼. 
  - 대부분의 요즘 컴퓨터는 64비트니까 걱정 마.
- 다운로드한 파일을 실행하고 나면, 계속 **'다음(Next)' 버튼만 누르면 돼.** 
  - 특별히 설정을 바꿀 필요 없이, 기본값으로 쭉 설치하면 돼. 
  - 마치 게임 설치할 때 '다음', '다음' 누르는 거랑 똑같다고 보면 돼. 

## 2. 깃허브 데스크톱(GitHub Desktop)은 깃(Git)이랑 달라 
<gen-svg-visual description="A Venn diagram showing the relationship between Git and GitHub Desktop. One circle is labeled 'Git' (the core tool for version control) and another overlapping circle is labeled 'GitHub Desktop' (a user-friendly interface for Git and GitHub). The overlapping area represents how GitHub Desktop uses Git. An arrow points from Git to GitHub Desktop, indicating GitHub Desktop relies on Git." />

- 깃(Git)은 컴퓨터에서 파일의 변화를 기록하고 관리하는 **핵심 도구**야. 
  - 마치 네가 숙제를 하면서 '1차 완성본', '2차 수정본' 이렇게 저장하는 것처럼, 깃은 코드의 모든 변화를 꼼꼼하게 기록해 줘.
- **깃허브 데스크톱(GitHub Desktop)**은 깃(Git)을 더 쉽게 쓸 수 있게 도와주는 **편리한 프로그램**이야. 
  - 깃(Git)을 직접 명령어로 쓰는 게 좀 어렵게 느껴질 수 있거든.
  - 깃허브 데스크톱은 그림(GUI)으로 보여줘서, 마우스 클릭 몇 번으로 깃(Git) 기능을 쓸 수 있게 해 줘. 
  - 마치 복잡한 자동차 엔진(Git)을 쉽게 조작할 수 있게 해주는 운전대와 페달(GitHub Desktop) 같은 거지.

## 3. 설치는 기본값으로 쭉! 


- 깃(Git)을 설치할 때 나오는 여러 설정 화면들은 대부분 건드릴 필요 없어. 
  - 그냥 **기본값(default)**으로 두고 '다음(Next)' 버튼만 계속 누르면 돼. 
  - 마치 새 스마트폰을 처음 켤 때 나오는 초기 설정들을 그냥 '확인'만 누르고 넘어가는 거랑 비슷해.
- 이렇게 하면 가장 일반적이고 문제없는 상태로 깃(Git)이 설치될 거야. 
  - 나중에 필요하면 설정을 바꿀 수도 있지만, 처음에는 기본이 최고야.

## 4. 설치 후에는 바로 사용 가능! 
<gen-svg-visual description="A simple diagram showing the flow after installation: Git Installation Complete -> Ready to Use. This visual emphasizes that no further complex steps are needed after the basic installation." />

- 설치가 끝나면 바로 깃(Git)을 사용할 준비가 된 거야. 
  - 따로 복잡한 설정이나 추가 작업이 필요 없어. 
  - 이제 너의 컴퓨터에서 깃(Git)을 이용해서 코드 관리를 시작할 수 있어!
