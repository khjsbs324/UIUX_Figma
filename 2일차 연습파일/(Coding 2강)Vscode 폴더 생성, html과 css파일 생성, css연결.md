Vscode에서 폴더와 파일을 어떻게 만들고 연결할까? Vscode에서 **폴더를 만들고 그 안에 **HTML**, **CSS** 파일을 생성한 뒤, HTML 파일에 CSS 파일을 연결하는 방법**을 알려줄 거야.
## 1. VS Code에서 작업 공간 만들기 
<gen-svg-visual description="A visual representation of creating a new folder in VS Code. It shows the VS Code interface with a 'New Folder' button highlighted, and a new folder named '1일차 연습 파일' being created in the file explorer panel." />

- **VS Code**는 코딩할 때 쓰는 프로그램인데, 여기서 파일을 정리할 폴더를 먼저 만들어야 해. 
  - 마치 책상 위에 책을 쌓아두는 대신, 책꽂이에 과목별로 정리하는 거랑 똑같아. 
- 폴더를 만들 때는 화면 오른쪽에 있는 '새 폴더' 버튼을 누르면 돼. 
  - 폴더 이름은 '1일차 연습 파일'처럼 알아보기 쉽게 지어주면 돼. 
- **팁**: 폴더가 제대로 만들어졌는지, 그리고 다른 폴더 안에 잘못 들어가지 않았는지 꼭 확인해야 해. 
  - 만약 폴더 모양이 이상하거나, 다른 폴더 안에 들어가 있으면 다시 만들어야 할 수도 있어. 

## 2. 폴더 안에 파일 정리하기 


- 폴더를 만들었으면, 이제 그 안에 HTML 파일이랑 CSS 파일을 넣어줄 거야. 
  - 마치 책꽂이에 과목별로 칸을 나눴으면, 이제 그 칸에 맞는 책들을 꽂아 넣는 거지.
- 파일을 폴더 안에 넣는 방법은 간단해. 
  - **HTML** 파일과 **CSS** 파일을 같이 선택한 다음, 마우스로 끌어서 '1일차 연습 파일' 폴더 안으로 옮기면 돼. 
  - 그러면 '옮길 거냐?'고 물어보는 창이 뜨는데, '옮긴다(Move)'고 선택하면 돼. 
- **팁**: 파일이 폴더 안에 잘 들어갔는지 확인하는 방법이 있어. 
  - 폴더 이름 옆에 있는 꺾쇠(>) 모양을 보면 돼. 
  - 꺾쇠가 아래를 향하고 있으면 폴더가 열려 있어서 안에 파일이 보이고, 오른쪽을 향하고 있으면 닫혀 있어서 파일이 안 보이는 거야. 
  - 파일이 폴더 안에 들어가 있으면, 파일 이름 앞에 빈 공간(들여쓰기)이 생겨서 한눈에 알아볼 수 있어. 

## 3. 새로운 **HTML**과 **CSS** 파일 만들고 연결하기 


- 이제 새로운 폴더('2일차 연습 파일')에 **HTML** 파일과 **CSS** 파일을 만들고, 서로 연결해 볼 거야. 
  - 마치 새 노트를 만들고, 그 노트에 맞는 표지 디자인을 연결하는 것과 같아.
- 먼저, 기존에 열려있던 파일들은 모두 닫아줘. 
  - 그래야 헷갈리지 않고 새 작업을 시작할 수 있어.
- '2일차 연습 파일' 폴더를 열고, 그 안에 두 가지 파일을 만들어. 
  - 첫 번째는 index.html이라는 **HTML** 파일이야. 
  - 두 번째는 style.css라는 **CSS** 파일이야. 
- 파일을 만들었으면, 이제 **HTML** 파일과 **CSS** 파일을 연결해야 해. 
  - **index.html** 파일을 열고, 느낌표(!)를 입력한 다음 Tab 키를 누르면 기본적인 **HTML** 코드가 자동으로 만들어져. 
  - 그 코드 안에 <link rel="stylesheet" href="style.css">라고 입력해서 **style.css** 파일을 연결해 주면 돼. 
  - 이렇게 하면 **HTML** 문서가 **style.css** 파일에 있는 디자인 규칙을 가져와서 사용하게 되는 거야.
