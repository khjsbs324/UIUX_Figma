
코딩할 때 검색 능력이 왜 중요할까? 새로운 코드가 계속 나오기 때문에, 혼자서 문제를 해결하고 필요한 정보를 찾아내는 능력이 코딩 실력의 95%를 차지할 정도로 중요해.
## 1. 코딩에서 검색 능력이 중요한 이유 
코딩은 마치 새로운 장난감을 계속 만드는 것과 같아. 새로운 장난감(코드)이 계속 나오기 때문에, 혼자서 문제를 해결하고 필요한 정보를 찾아내는 능력이 정말 중요해.

- **혼자서 문제 해결하는 능력**: 코딩 실력의 95%를 차지할 정도로 중요해. 
  - 수업에서 배우는 건 5% 정도밖에 안 돼. 
  - 디자인은 선생님이 95%를 해주고 네가 5%만 하면 되지만, 코딩은 반대야. 
- **정보를 얻는 유일한 방법**: 검색을 못 하면 필요한 정보를 얻을 수 없어. 
  - 정보가 없으면 혼자 공부할 수도 없지. 
- **새로운 코드의 등장**: 코딩 세계에서는 새로운 코드가 계속해서 만들어져. 
  - 모든 코드를 다 배울 수는 없으니, 필요할 때마다 찾아내는 능력이 필수적이야. 

## 2. justify-content: space-between으로 요소 정렬하기 

justify-content: space-between은 여러 개의 물건(요소)을 일정한 간격으로 나란히 놓을 때 쓰는 마법 주문 같은 거야.

- **어떤 상황에 쓸까?**:
  - 헤더(웹사이트 맨 위 부분)에 로고나 메뉴 같은 여러 요소를 왼쪽, 가운데, 오른쪽에 딱 맞춰서 배치하고 싶을 때 사용해. 
  - 예를 들어, 왼쪽에는 회사 로고, 가운데에는 제목, 오른쪽에는 로그인 버튼을 놓는 것처럼 말이야.
- **어떻게 작동할까?**:
  - 이 주문은 플렉스**(flex)**라는 특별한 공간(컨테이너) 안에서만 작동해. 
  - space-between을 사용하면, 첫 번째 물건은 맨 왼쪽에, 마지막 물건은 맨 오른쪽에 딱 붙고, 나머지 물건들은 그 사이 공간에 똑같은 간격으로 떨어져서 배치돼. 
  - 마치 세 명의 친구가 일렬로 서 있는데, 양 끝 친구는 벽에 딱 붙고, 가운데 친구는 나머지 공간에 맞춰서 서는 것과 같아.
- **사용 방법**:
  - 요소를 담고 있는 부모 태그(예: <header>)에 display: flex;를 먼저 적용해야 해. 
  - 그리고 그 부모 태그에 justify-content: space-between;을 추가하면 돼. 

## 3. 코딩 문제 해결을 위한 검색 팁

코딩 문제를 만났을 때, 정답을 바로 찾기보다는 어떻게 검색해서 해결하는지 배우는 게 중요해.

- **핵심 키워드 찾기**: 검색할 때는 어떤 단어를 써야 할지 아는 게 중요해. 
  - 예를 들어, "플렉스"와 "같은 여백"처럼 문제에서 힌트를 주는 단어를 사용해봐. 
  - 시각적으로 보이는 형태(예: 왼쪽, 가운데, 오른쪽 정렬)도 좋은 키워드가 될 수 있어. 
- **검색 결과 분석**: 검색하면 여러 가지 코드가 나올 거야. 
  - 그중에서 네가 원하는 결과와 **비슷한 형태**를 만들어주는 코드를 찾아야 해. 
  - 모든 내용을 다 읽기보다는, 필요한 부분을 빠르게 파악하는 연습을 해야 해. 
- **검색 능력의 중요성**:
  - 검색 능력은 자바스크립트 같은 더 어려운 코딩을 할 때도 혼자서 문제를 해결하는 데 필수적이야. 
  - 초반에 이 능력을 키우지 않으면 나중에 혼자 공부하기가 정말 어려워질 거야.
