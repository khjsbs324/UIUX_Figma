## 1. 피그마(Figma)와 VS Code(비주얼 스튜디오 코드) 연결 준비 
피그마와 VS Code를 연결하려면 몇 가지 준비가 필요해. 마치 친구 집에 놀러 가기 전에 옷을 갈아입고 준비물을 챙기는 것과 같아.


- 디브 모드**(Dev Mode) 켜기** 
  - 피그마에서 '디브 모드'라는 특별한 모드를 켜야 해. 이건 개발자들이 디자인을 코드로 옮길 때 쓰는 모드라고 보면 돼.
- **내 **프레임**(Frame) 확인하기** 
  - 다른 사람이 만든 프레임 말고, 네가 직접 만든 프레임에서 작업을 해야 해. 마치 네 방에서 네 물건을 정리하는 것처럼 말이야.
- **불필요한 탭(Tab) 닫기** 
  - 작업하다 보면 여러 창이 열려서 복잡해질 수 있어. 필요한 창만 남기고 나머지는 닫아서 깔끔하게 정리하는 게 좋아.

## 2. 피그마에서 깃허브(GitHub) 플러그인(Plugin) 연결하기 
피그마에서 깃허브에 있는 파일을 불러오려면 '플러그인'이라는 추가 기능을 설치하고 연결해야 해. 이건 마치 스마트폰에 필요한 앱을 설치하는 것과 비슷해.


- **깃허브 플러그인 찾기 및 실행** 
  - 피그마 오른쪽 메뉴에서 '플러그인' 탭을 눌러.
  - 이미 깃허브 플러그인이 있다면 점 세 개를 눌러 '뷰 디테일(View Detail)'을 선택하고 '런(Run)'을 눌러 실행해.
  - 만약 없다면 검색창에 '깃허브'라고 검색해서 찾은 다음, 아이콘을 확인하고 열어줘.
- 프레임** 선택 및 **URL**(주소) 입력 준비** 
  - 깃허브 플러그인이 열리면, 네가 작업할 '프레임'을 선택해야 해.
  - 프레임을 선택하면 URL(인터넷 주소)을 입력하라는 창이 뜰 거야. 여기에 깃허브에 있는 파일 주소를 넣어줄 거야.

## 3. 깃허브에 HTML(웹 페이지 파일) 파일 올리기 
피그마에서 불러올 HTML 파일이 깃허브에 있어야 해. 이건 마치 네가 만든 그림을 친구에게 보여주기 위해 클라우드(인터넷 저장 공간)에 올리는 것과 같아.


- 깃허브** 데스크톱(Desktop) 앱 열기** 
  - 깃허브 데스크톱 앱을 열어서 네가 만든 HTML 파일이 잘 있는지 확인해.
- **파일 **커밋**(Commit) 및 **푸시**(Push)하기** 
  - 만약 파일이 없다면, VS Code에서 만든 5일차 파일을 깃허브에 올려야 해.
  - '커밋(Commit)'은 변경 내용을 저장하는 거고, '푸시(Push)'는 저장된 내용을 깃허브 서버로 보내는 거야.
- **깃허브에서 파일 확인 및 링크(Link) 복사** 
  - 깃허브 웹페이지에서 새로고침을 해서 5일차 연습 파일이 잘 올라왔는지 확인해.
  - 파일이 보이면 해당 폴더를 누르고, 'index.html' 파일을 클릭해.
  - 그러면 파일의 '링크(URL)'가 보일 거야. 이 링크를 복사해 둬.



## 4. 피그마와 깃허브 파일 최종 연결 및 VS Code에서 확인 
이제 복사한 깃허브 파일 링크를 피그마에 붙여넣고, VS Code에서 디자인을 바로 확인할 수 있도록 연결할 거야. 이건 마치 친구에게 보낸 그림 파일 링크를 받아서 바로 열어보는 것과 같아.


- **피그마에 깃허브 링크 붙여넣기** 
  - 피그마로 돌아와서 아까 열어둔 깃허브 플러그인 창에 복사한 링크를 붙여넣어.
  - '링크 투 레이어(Link to Layer)' 버튼을 클릭하면 파일이 로딩될 거야.
  - 만약 잘 안 되면, 피그마에서 '프레임'을 다시 한번 클릭하고 링크를 붙여넣어 봐.
- **VS Code에서 피그마 디자인 확인** 
  - 피그마 플러그인 창 왼쪽 탭에서 '깃허브' 모양을 찾아.
  - 점 세 개를 누르면 '오픈 인 VS 코드(Open in VS Code)'라는 메뉴가 보일 거야. 이걸 클릭해서 VS Code를 열어줘.
  - VS Code 화면 오른쪽에 피그마 디자인이 뜨는지 확인해.
- **로그인(Login) 문제 해결** 
  - 만약 VS Code에서 피그마 디자인이 안 보인다면, 피그마 로그인이 안 되어 있을 수 있어.
  - VS Code 왼쪽 메뉴에서 피그마 아이콘을 누르고 로그인 버튼을 눌러서 다시 로그인해 줘.
  - 로그인 후에는 네 계정으로 잘 연결되었는지 확인해야 해.
- 로그아웃**(Logout) 방법** 
  - 로그아웃하고 싶을 때는 점 세 개를 누르면 로그아웃 메뉴가 보일 거야.
