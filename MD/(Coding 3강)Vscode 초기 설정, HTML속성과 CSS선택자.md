
코딩할 때 HTML 속성과 **CSS 선택자**를 어떻게 써야 할까? HTML 속성은 여는 태그 안에 쓰고, CSS 선택자는 태그 선택자, 클래스 선택자, 아이디 선택자 세 가지가 있는데, 특히 **클래스 선택자**는 여러 번 쓸 수 있는 별명이고, **아이디 선택자**는 딱 한 번만 쓸 수 있는 고유한 이름이야.
## VS Code 초기 설정 및 HTML 속성, CSS 선택자 배우기


## 1. VS Code 기본 설정하기 
<gen-svg-visual description="A diagram showing the VS Code settings menu with arrows pointing to font size, font weight, line height, and cursor thickness settings. It also highlights the theme selection for color themes." />
코딩을 편하게 하려면 **VS Code** (비주얼 스튜디오 코드)라는 프로그램을 내 눈에 맞게 설정해야 해.

- **한글판 사용은 비추천** 
  - 자격증 시험처럼 꼭 필요할 때만 잠깐 쓰는 게 좋아 
- **기본 설정 메뉴 찾기** 
  - 왼쪽 위에 있는 '파일(File)' 메뉴를 누르고, '기본 설정(Preferences)'에서 '설정(Settings)'을 선택하면 돼.
  - 단축키로는 Ctrl + 콤마(,)를 눌러도 바로 열 수 있어 
- **글자 크기 조절** 
  - 눈이 좋으면 16으로, 조금 나쁘면 18, 많이 나쁘면 21이나 24로 설정하면 글자가 잘 보여 
- **글자 두께 조절** 
  - 검색창에 font-weight (폰트 웨이트)라고 검색해서 '노멀(normal)'을 '볼드(bold)'로 바꾸면 글자가 두꺼워져서 눈이 덜 아파 
- **줄 간격 조절** 
  - 검색창에 line-height (라인 하이트)라고 검색해서 줄 간격을 1.7로 설정하면 글자들이 너무 붙어있지 않아서 읽기 편해 
- 커서 두께** 조절** 
  - 검색창에 width (위스)라고 검색해서 커서 두께를 4 정도로 설정하면 커서가 잘 보여서 어디에 글자를 쓰고 있는지 알기 쉬워 
- **마우스 올렸을 때 뜨는 팝업 끄기** 
  - 마우스를 올리면 뜨는 작은 창(팝업)들이 귀찮을 수 있어. hover (호버) 설정을 '온(on)'에서 '오프(off)'로 바꾸면 돼 
- 테마**(**색상**) 변경** 
  - '파일(File)' > '기본 설정(Preferences)' > '테마(Theme)' > '색 테마(Color Theme)'에서 '모노카이(Monokai)' 같은 눈이 편한 테마를 고르면 돼 
  - 눈이 덜 아프고 코드 구분이 쉬워져 
- **자동 완성 기능 끄기** 
  - 수업 중에 자동 완성 기능이 방해될 수 있으니, 필요하다면 이 기능을 꺼두는 게 좋아 
- 탭**(Tab) 크기 설정** 
  - 코드를 들여쓰기 할 때 쓰는 탭 크기는 보통 두 칸이지만, 처음에는 네 칸으로 설정해서 코드를 구분하기 쉽게 하는 게 좋아 

## 2. HTML 속성: 태그에 추가 정보 주기 
<gen-svg-visual description="A diagram illustrating HTML attributes. It shows an opening HTML tag like <div class='my-class'>. The 'class' is highlighted as the attribute name and 'my-class' as the attribute value. It emphasizes that attributes are written inside the <keyword data-source-id=" />
**HTML 속성** (에이치티엠엘 속성)은 HTML 태그에 특별한 정보를 추가하는 방법이야. 마치 옷에 주머니나 단추를 다는 것과 같아.

- **속성은 어디에 쓸까?** 
  - 속성은 **여는 **태그 안에 써야 해. 닫는 태그나 태그 바깥에는 쓰지 않아.
  - 마치 밥그릇 안에 밥을 담듯이, 태그 안에 속성을 넣는다고 생각하면 돼 
- **속성 쓰는 방법** 
  - 태그 이름 뒤에 **한 칸 띄고** 속성 이름을 써.
  - 예를 들어, div 태그에 class (클래스)라는 속성을 추가하려면 div class 이렇게 쓰는 거야 
- **속성 값 주기** 
  - 속성 이름 뒤에는 = (등호)를 쓰고, 그 뒤에 쌍따옴표** **"" 안에 속성 값을 넣어줘.
  - 예를 들어, class="my-box" (클래스 이퀄 마이 박스)처럼 쓰는 거지.
  - **따옴표 모양 주의**: 작은따옴표 '' 말고 꼭 쌍따옴표 ""를 써야 해 

## 3. CSS 선택자: 꾸며줄 대상을 고르는 방법 

**CSS 선택자** (씨에스에스 선택자)는 HTML 문서에서 내가 꾸며주고 싶은 부분을 정확히 골라내는 방법이야. 마치 반에서 특정 친구를 부르는 방법과 같아.

- **1. **태그 선택자 
  - **본명으로 부르기**: HTML 태그 이름(예: div, p, span)을 그대로 써서 선택하는 방법이야.
  - **모두에게 적용**: 이 태그를 가진 모든 요소에 스타일이 적용돼. 예를 들어, div라고 쓰면 모든 div 태그가 똑같이 꾸며지는 거지 
  - **CSS에서 쓰는 법**: 그냥 태그 이름을 써. div { ... }
- **2. **클래스 선택자 
  - **별명으로 부르기**: 여러 친구가 함께 쓰는 별명처럼, 여러 HTML 요소에 같은 별명(클래스)을 붙여서 선택하는 방법이야.
  - **여러 번 사용 가능**: 이 별명은 여러 요소에 반복해서 붙일 수 있어. 예를 들어, '공주'라는 별명을 여러 친구에게 붙여줄 수 있는 것과 같아 
  - **HTML에서 쓰는 법**: 여는 태그 안에 class="별명" (클래스 이퀄 별명)이라고 써.
  - **CSS에서 쓰는 법**: 별명 앞에 **점 **.을 붙여. .별명 { ... }
- **3. **아이디 선택자 
  - **고유한 이름으로 부르기**: 딱 한 명만 쓸 수 있는 고유한 이름처럼, HTML 문서 전체에서 **단 한 번만** 사용할 수 있는 이름(아이디)을 붙여서 선택하는 방법이야.
  - **한 번만 사용**: 연인끼리만 쓰는 애칭처럼, 이 이름은 다른 요소에 절대 붙일 수 없어 
  - **HTML에서 쓰는 법**: 여는 태그 안에 id="고유이름" (아이디 이퀄 고유이름)이라고 써.
  - **CSS에서 쓰는 법**: 고유이름 앞에 **샵 **#을 붙여. #고유이름 { ... }

## 4. 주석: 코드에 메모 남기기 
<gen-svg-visual description="A simple diagram showing a code snippet with a comment. The comment is highlighted and explained as 'ignored by the browser, for developers only'. It shows how to write a single-line comment using '//' or '<!-- -->' in HTML/CSS context." />
**주석** (주석)은 코드에 메모를 남기는 기능이야.

- **개발자들끼리 보는 메모**: 주석으로 쓴 내용은 웹 브라우저에 나타나지 않고, 코드를 보는 개발자들만 볼 수 있어 
- **사용 목적** 
  - **메모**: 코드에 대한 설명을 남겨서 나중에 다시 봤을 때 쉽게 이해할 수 있게 해줘.
  - **협업**: 다른 개발자와 함께 작업할 때 서로의 코드를 이해하는 데 도움을 줘.
  - **기록**: 코드의 변경 사항이나 중요한 정보를 기록해둘 수 있어.
- **주석 쓰는 방법** 
  - Ctrl + 슬래시(/)를 누르면 주석 처리가 돼.
  - CSS에서는 /<em> 주석 내용 </em>/ 이렇게 써.

  - 여러 번 반복해서 사용할 수 있는 건 클래스 선택자 (class selector)야.
