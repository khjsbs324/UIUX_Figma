
코딩에서 섹션 태그는 무엇이고 왜 사용할까? 섹션 태그는 웹페이지의 내용을 **의미 있는 덩어리(공간)**로 나누는 역할을 해. 마치 쇼핑몰에서 '상의', '하의'처럼 카테고리를 나누는 것과 같아서, 웹페이지를 깔끔하게 정리하고 관리하기 쉽게 만들어주는 중요한 도구야.
## 1. 웹페이지를 정리하는 섹션 태그 

- 웹페이지를 만들 때, 내용을 의미 있는 덩어리(공간)로 나누는 역할을 하는 게 바로 **섹션 태그**야. 
  - 마치 쇼핑몰에서 '상의', '하의', '아우터'처럼 카테고리를 나누는 것과 같다고 보면 돼. 
  - 이렇게 나누면 웹페이지를 깔끔하게 정리하고 관리하기 쉬워져. 
- **섹션 태그**는 자동으로 줄 바꿈이 돼서, 각 섹션은 한 줄에 하나씩 나타나. <gen-svg-visual description="A visual representation of a webpage divided into multiple sections. ## 2. 코딩할 때 키보드만 쓰는 연습 

- 코딩 속도를 높이려면 마우스 대신 키보드를 사용하는 연습이 중요해. 
  - 시프트**(Shift) **키: 텍스트를 선택할 때 사용해. 커서가 움직이는 방향대로 글자를 선택할 수 있어. 
  - 홈**(Home) 키**: 커서를 현재 줄의 맨 앞으로 이동시켜줘. 
  - **엔드(End) 키**: 커서를 현재 줄의 맨 뒤로 이동시켜줘. 
  - 컨트롤**(Ctrl) 키**: 단어 단위로 빠르게 이동하거나, 복사(Ctrl+C), 붙여넣기(Ctrl+V)를 할 때 사용해. 
- **키보드 활용 예시**:
  - 한 줄을 통째로 복사하려면: 커서를 줄 끝에 두고 Shift + Home 키를 누른 다음, Ctrl + C를 눌러. 
  - 여러 줄을 빠르게 붙여넣으려면: Ctrl + V를 누르고 엔터(Enter) 키를 반복해서 눌러줘. 


## 3. 웹페이지 높이 설정하기: 뷰포트 하이트**(VH)** 

- 웹페이지에서 섹션의 높이를 설정할 때는 **뷰포트 하이트(Viewport Height, VH)** 단위를 사용할 수 있어. 
  - **100VH**는 화면 전체 높이만큼을 의미해. 
  - 이렇게 설정하면 어떤 기기에서 보든 화면 크기에 맞춰서 섹션 높이가 조절돼. 


## 4. 웹에서 색상 표현하기 

- 웹에서 색상을 표현하는 방법은 여러 가지가 있지만, 주로 헥사 코드**(Hex Code)**를 많이 사용해. 
  - **헥사 **코드는 # 뒤에 6자리의 숫자와 알파벳(0-9, A-F)을 조합해서 색을 나타내는 방식이야. 
  - RGB (빨강, 초록, 파랑)나 CMYK (청록, 자홍, 노랑, 검정) 같은 방식도 있지만, 웹에서는 헥사 코드가 더 간편해서 많이 쓰여. 
  - **CMYK**는 주로 인쇄할 때 사용하는 방식이야. 
- 색상** 선택 도구(Color Picker)**를 사용하면 원하는 색상의 헥사 코드를 쉽게 찾을 수 있어.
