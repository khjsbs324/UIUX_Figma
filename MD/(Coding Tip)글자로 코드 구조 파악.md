## 1. 코딩 시 글자로 코드 구조 파악하기 
<gen-svg-visual description="This visual illustrates how to understand code structure by using text. It shows a screen divided into sections like 'Navigation Bar', 'Logo', and 'User Menu', each labeled with its corresponding text to easily identify and differentiate code blocks." />
코딩할 때 화면에 직접 글자를 작성하여 코드의 구조를 쉽게 파악할 수 있다.


### 1.1. 글자를 활용한 코드 구조 구분 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/ZM4uI5Y6A2I/15.jpg)
1. 글자** 작성의 필요성** 
  1. 현재 화면에 아무것도 없으면 코드 구조를 구분하기 어렵다 
2. **글자 **작성** **방법 
  1. 내비게이션 바 영역에는 "내비게이션 바"라고 작성한다 
  2. 로고 영역에는 "로고"라고 작성한다 
  3. 사용자 메뉴 영역에는 "사용자 메뉴"라고 작성한다 
  4. 각 영역에 해당하는 글자를 직접 작성하여 구조를 명확히 구분한다
