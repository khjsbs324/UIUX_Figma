- 여러 명이 한 컴퓨터에서 **깃허브 데스크톱 앱**을 같이 쓸 때는 조심해야 할 게 있어.

## 2. 내 이름과 이메일로 설정하는 방법 


- 누가 작업했는지 정확히 기록하려면, **깃(Git)** 설정에서 **이름**과 **이메일**을 꼭 자기 걸로 바꿔줘야 해. 
  - **파일(File)** 메뉴에서 **옵션(Options)**으로 들어가 봐. 
  - 거기서 **깃(Git)**이라는 항목을 찾을 수 있을 거야. 
  - **이메일(Email)**이 '다른 사람(Other)'으로 되어 있거나 다른 친구 이메일로 되어 있을 수도 있어. 
  - 이걸 반드시 **네 이메일**로 바꿔줘야 해. 
  - 로그인 정보도 네 것인지 확인하는 게 좋아. 
- 이렇게 설정하지 않으면, 네가 작업한 내용이 다른 사람 이름으로 기록될 수 있어. 
  - 마치 네가 쓴 일기장에 다른 친구 이름이 적히는 것과 같아. 
  - 나중에 누가 어떤 작업을 했는지 헷갈려서 문제가 생길 수 있겠지?
